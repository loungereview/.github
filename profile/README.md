# LoungeReview — Product, App & API Overview

LoungeReview is a global platform dedicated to helping travelers discover, access, and evaluate airport lounges worldwide. We also offer the ability to pre-book and purchase access to hundreds of lounges.
The LoungeReview API is available to qualified partners and developers.

---

## ✈️ About LoungeReview

LoungeReview maintains the most comprehensive and continuously updated airport lounge datasets in the world.  
Our team combines expert curation, on‑site reviews, and community contributions to deliver accurate, structured information for travelers and enterprise partners.

- Thousands of airport lounges across the globe  
- Detailed access rules for airlines, alliances, elite status, credit cards, and memberships  
- Daily updates to lounge data, photos, and access policies  

Visit us at: https://loungereview.com

## 🧩 LoungeReview API

The LoungeReview API provides structured, high‑quality airport lounge data for enterprise travel applications.  
It is designed for reliability, accuracy, and scale.

Find out more at: https://loungereview.com/api/

### Core Endpoints

#### **Airport API**
Retrieve airport metadata:
- Name  
- IATA code  
- City & country  
- GPS coordinates  

#### **Lounge Search API**
Search lounges by:
- Airport  
- Airline  
- Alliance  
- Operator  
- Membership program  
- Credit card  

#### **Lounge Details API**
Fetch structured lounge information:
- Opening hours  
- Amenities  
- Food & beverage details  
- Access rules  
- Photos (CDN‑optimized)  
- User comments  
- Contact information  

#### **Lounge Access API**
Determine access eligibility for any itinerary based on:
- Airline & class of service  
- Frequent flyer status  
- Lounge memberships  
- Premium credit cards  
- Active military status  
- Origin/destination airport
